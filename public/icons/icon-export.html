Instructions for exporting icons:

  1. Take a screenshot of each icon above
  2. Crop the screenshot to the dimensions of the icon itself
  3. Save each icon with its corresponding name in the /public/icons/ directory
  4. Make sure the file dimensions match their names (512x512, 192x192, 180x180)
  5. For creating favicon.ico, use any online converter to transform icon-512x512.png into a multi-size ico file
  6. Place favicon.ico in the /public/ directory
